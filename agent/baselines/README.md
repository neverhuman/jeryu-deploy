# Jankurai ratchet baselines

There is intentionally no active accepted baseline in this directory until a
fresh protected-main Jankurai 1.6.11 score receives detached review. The
superseded baseline is retained byte-for-byte under `historical/` as
non-authoritative audit history; it must never be relabelled as 1.6.11 evidence.

Create `main.repo-score.json` only from a fresh protected-main run after the
identity-locked Jankurai 1.6.11 binary is installed, then subject that exact
baseline to detached review before enabling the ratchet gate.
